# Orchipelago!3

Orchipelago!3 is a mod for Orcs Must Die! 3 (OMD3).  
It randomizes items through Archipelago.

### Items Randomized
- Traps
- Weapons
- Characters
- Trinkets
- Upgrades
- Mission Unlocks
- Endless Maps

### Checks
- Completing any level
- Completing any level with 5 skulls
- Completing 25 waves of Endless
- Completing a level with the 'Untouchable' bonus (complete a level without taking damage)

### DLC?
The mod will feature support for every DLC. 
You will have the choice of which Campaigns to include in the run.

### Goal
There is currently one customizable goal; simply complete every campaign. You can choose which of the four available to include, as well as whether you want to include completing Endless and Scramble.
- [ ] Old Friends
- [ ] Drastic Steps
- [ ] Cold as Eyes
- [ ] Tipping the Scales
- [ ] Endless
- [ ] Scramble

Simply put an x in the campaigns you wish to include.
